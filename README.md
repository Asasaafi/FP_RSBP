# FP_RSBPP

## Requirements

### Install dependencies berikut sebelum menjalankan project:

``pip install pandas``

``pip install neo4j``

``pip install nltk``

``pip install streamlit``

``pip install plotly``

``pip install pyarrow``

## Run
``python -m streamlit run 1_cleaning_data.py``

``python -m streamlit run 2_import_to_neo4j.py``

``python -m streamlit run 3_sentiment_analysis.py``

``python -m streamlit run 4_dashboard.py``
